Fault detection in industrial processes project

The matlab file (m file) requires two packages (center and mavarge) from the datana 1.1.3 toolbox.
The pca used in this project is that of matlab built-in function.
Other functions used are also matlab native functions.

The PracticalAD.m file contains all the executable code for this project and 
the ADAML_report contains the analysis of the project.
